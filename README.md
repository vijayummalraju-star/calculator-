This is a **Progressive Web App (PWA)** which works just like an APK on Android!

### Method 1: Install from Browser (Recommended)

1. Open the `index.html` file in **Google Chrome** or **Microsoft Edge** on your phone
2. Tap the **menu button** (three dots) in the browser
3. Select **"Add to Home screen"** or **"Install app"**
4. The app will be installed like a regular app!
5. You can now launch it from your home screen

### Method 2: Use a Web Server

1. Upload all files to a web server or use a service like:
   - GitHub Pages (free)
   - Netlify (free)
   - Vercel (free)
   
2. Visit the URL on your phone
3. Follow the install prompt
